About the homologyScanner Web Tool

The homologyScanner web server was started as a project for a LIMS course at Uppsala university (Sweden). The members of the development team consisted of Fredrik Wrede, Anastasios Glaros and Athanasios Alexiou, students in the master programme in bioinformatics. The project was started during the first spring period of 2015, and contiued through 2018 by Anastasios Glaros, Athanasios Alexiou and Samuel Flores. The homologyScanner method itself, and its back-end implementation, were written by Samuel Flores.

Commercial use

This web server is a public resource with no privacy options. If you want a private homologyScanner server set up behind your company firewall, we will be pleased to help. Contact samuel.flores@scilifelab.se for information.
